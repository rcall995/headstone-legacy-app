Laser Etching Guide

Recommended Sizes
  • Minimum: 1" x 1" (25mm x 25mm)
  • Recommended: 1.5" x 1.5" (38mm x 38mm)
  • Large/Distance: 2" x 2" (50mm x 50mm)
Best Practices
  • Use high contrast (black on light stone)
  • Include quiet zone (white border)
  • Test scan before final etch
